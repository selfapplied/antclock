## Attribution Requirements

Under the CC BY-SA 4.0 license, you must:

1. **Give appropriate credit** - Include the citation above
2. **Indicate changes** - If you modify the work, state the changes made
3. **Share alike** - Distribute contributions under the same license
4. **Link to license** - Provide a link to http://creativecommons.org/licenses/by-sa/4.0/
